발주요청은 사이트에서도 가능하도록 기능추가
엑셀 업로드 기능
발주요청 리스트는 타입별로 구분해서 볼 수 있도록 수정
관리자는 요청받은 구매요청 항목을 체크해서 없앨 수 있어야함
 - 발주를 한경우 리스트에서 사라져야함
약품관리의 카테고리를 수정할 수 있는 기능이 있어야함(ex:향정신성 약품 이런걸 추가또는 삭제 수정이 가능해야함)
모바일에서 발주요청시 이미 발주요청 상태를 확인해서 동일한게 요청되어 있는상태인지 모바일에서 알려주자
발주여부가 승인상태로 바뀌면 리스트에서 보이면 안됨(승인된 리스트만 볼 수 있도록 수정해야함:필터)
발주요청 월단위로 볼 수 있어야 함
========

15일(월)부터 법인 정기감사
 - 다다음주 월요일부터는 신경을 쓰기 어려움
